# EchartsDemo
百度EChart在Android上的demo（为百度EChart的开发者们点赞，真的是大神的杰作啊！）<br>
采用guozilan大神的abel533/ECharts针对ECharts3.x(2.x)版本的Java类库，实现在Android上Echarts的demo，包括柱状图，折线图和饼状图，分别展示了从JS文件加载数据和用JAVA代码组装数据。<br>
